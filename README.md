# DeepCache_Efficient_ML_CAP6614
Code for CAP6614 Efficient ML delving into DeepCache and comparing it to other caching algorithms

### Running Code
For each folder, in order to reproduce our results you first need to make sure you have all dependencies installed. This can be done in 2 ways:
1. Create a virtual environment and install the dependencies:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
2. Load the python notebooks in google colab and run cells to download libraries as needed.
   
Once this is done you can run the cells in the folder's python notebook to reproduce our results.
